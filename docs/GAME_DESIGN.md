## 一、游戏定位

「仙侠开放世界」× 「宝可梦式角色养成」× 「雪中悍刀行式叙事」

### 世界观核心
- **九霄龍脈 / 遺墨**：主角体内封印的禁忌皇权之力
- **故事起点**：前朝覆灭，幼年太子被国师墨綰青封印记忆隐居小村。十六年后追兵到来，主角逃往武当山，开启江湖之路
- **剧情类型**：
  - **不可重复剧情**：主线剧情（章节推进）、支线剧情（角色个人线）
  - **可重复剧情**：日常任务（砍柴、挑水等）、随机任务（江湖偶遇、突发事件）

## 二、修为境界体系

> 每个大境界分 **10 个小层**（如炼气一层~炼气十层）。突破大境界需完成试炼任务。
> 等级数值 `level` 从 0 开始：0=凡人，1=炼气一层，2=炼气二层……10=炼气十层，11=筑基一层……

| 大境界 | level范围 | 小层 | 身份 | 突破条件 |
|--------|-----------|------|------|----------|
| **凡人** | 0 | 0层 | 未入门 | — |
| **炼气** | 1~10 | 1~10层 | 外门弟子 | 入武当自动突破 |
| **筑基** | 11~20 | 1~10层 | 内门弟子 | 通过外门试炼考 |
| **结丹** | 21~30 | 1~10层 | 真传弟子 | 通过内门晋升试炼 |
| **元婴** | 31~40 | 1~10层 | 长老 | 结丹圆满 + 门派贡献 |
| **化神** | 41~50 | 1~10层 | 掌门级 | 元婴圆满 + 天地感悟 |
| **渡劫** | 51~60 | 1~10层 | 入圣 | 化神圆满 + 渡天劫 |
| **大乘** | 61~70 | 1~10层 | 传说 | （未知） |
| **飞升** | 71~80 | 1~10层 | 仅存典籍 | （未知） |

### 经验曲线

```
每小层所需经验 = 50 × 层数^1.5
炼气一层:50  二层:141  三层:260  四层:400  五层:559
六层:735  七层:926  八层:1131  九层:1350  十层:1581
```

> 大境界第十层满后不会自动突破，需完成对应试炼。

## 三、人物设定

### 主角
前朝唯一真龍太子（名字自取），记忆被封印，以为自己是普通少年。体内潜藏九霄龙脉。

### 四大女主

| 女主 | 故事类型 | 情感基调 |
|------|----------|----------|
| **柳清寒** | 武当师姐·命中夫妻 | 纯粹·清冷·暗涌 |
| **沈霓裳** | 江湖历练·正邪斗争 | 刚烈·不服→臣服 |
| **赵沁微** | 朝堂·身世觉醒 | 虐心·隐忍·归来 |
| **墨绐青** | 冲师·记忆回收 | 守护·落子·无悔 |

> 详细人物设定见各章节脚本文件（`docs/chapters/`）。

### 叙事节奏
1. **武当期**（第1~3章）：武当练武成长，柳清寒线
2. **江湖期**（第4~6章）：下山历练，沈霓裳线
3. **入世期**（第7章起）：身世面对，赵沁微+墨绐青线
4. **结局期**：帝王归位

| 🧹 打扫大殿 | 1时辰 | 经验+12, 铜钱+4 | 外门弟子（chapter≥2） |

| 🌙 后山老松树修炼 | 2时辰 | 经验+30 | 炼气二层（level≥2） |

> 小概率事件：砍柴遇宋知远偷懒（好感+1）、挑水遇顾小桑（获小道消息）、打扫遇张玄素（随机对话）、抄经遇陈静虚（练功心得+10经验）。

#### 主线剧情触发区（下方）

根据当前章节和act显示对应营地场景，点击按钮触发主线剧情VN或战斗。

### 人物关系面板

- **可折叠分类**：🌸 女主角 / ☯️ 武当派 / 👤 其他
- 每人显示 **9:16 小立绘** + 好感度数值
- **未解锁角色**：灰色立绘 + "未解锁"标注（如趙沁微在第三章前未解锁）
- 随剧情推进逐步解锁（沈霓裳在第二章act≥4后解锁）

## 五、战斗系统

### 伤害公式
```
伤害 = max(1, ATK × powerMul - DEF × defPen) × [0.9,1.1] × (暴击?1.5:1)
```

### 状态效果
中毒/剧毒/眩晕/减防/攻强/防强/闪避/回蓝
