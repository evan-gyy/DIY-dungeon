## Act 1：归途·消息

### 场景：武当山·紫霄宫门前·清晨

**【VN节点·归山】**

襄阳城一行后，五人御剑返回武当山。晨雾缭绕，太和宫的金顶在云海中若隐若现。

```
【陈静虚】站在紫霄宫门前，转身看向众人。
    "襄阳之事，我已向掌门禀报。黑月教在江南一带活动频繁，此事需从长计议。"
    "你们先下去休息，明日辰时，掌门要亲自召见。"
```

五人行礼散去。陈静虚看着主角的背影，眼中闪过一丝复杂神色。

```
【陈静虚】（低声）"这小子……身上的气息，又变强了。"
```

### 场景：武当山·内门厢房·傍晚

**【VN节点·师姐夜访】**

主角正在房中打坐调息，忽然门外传来熟悉的清冷声音。

```
【柳清寒】推门而入，手中捧着一个玉瓶。
    "师弟，掌门赐下的筑基丹，你服用了吗？"
    "此丹需在突破筑基时配合使用，方能将根基打得更为牢固。"
```

主角起身行礼，接过玉瓶。

```
【主角】（陆季喆）"多谢师姐关心，师弟尚未服用。"
【柳清寒】"嗯。"
    她走近几步，目光在主角身上停留片刻。
    "你身上的气息，比离开武当时浑厚了许多。"
    "突破筑基……应当就在这两日了。"
```

柳清寒忽然伸出手，指尖点在主角眉心。一股清冷的灵力探入体内，又迅速收回。

```
【柳清寒】眉头微蹙。
    "你体内的封印……似乎又松动了几分。"
    "墨绐青留下的手段，我无法窥探。但你要小心，封印松动往往伴随着危险。"
    "若有任何异样，立刻来找我。"
```

她转身欲走，又停下脚步。

```
【柳清寒】"对了，掌门明日召见，是为了宣布一件事。"
    "外门试炼已过，内门晋升试炼将在三个月后举行。"
    "届时，你需要代表武当参与五派论剑。"
    "……你若能夺得名次，对柳氏一族的处境，会有很大帮助。"
```

她的声音平淡如常，但主角敏锐地察觉到其中隐藏的某种期待。

```
【柳清寒】"早些休息。"
    她踏出门槛，身影消失在夜色中。
```

**【好感度事件完成】** 柳清寒好感 +3（当前好感：85）
> *柳清寒深夜送丹，透露五派论剑之事，对主角寄予厚望。*

**【VN节点·月下突破】**

夜深人静，主角独自来到后山老松树下。

他盘膝而坐，取出玉瓶中的筑基丹服下。丹药入腹，一股温热之力瞬间游走全身经脉。

```

突破目标：筑基一层（level 11）

与此同时，他体内那沉睡已久的封印开始躁动。

```
【旁白】
那封印了十六年的力量，此刻如同被唤醒的巨龍，
在体内翻涌咆哮，与筑基丹的药力交织碰撞。
```

一股前所未有的力量从丹田升起，冲向四肢百骸。

```
【主角】（内心）
这是……九霄龙脉的力量？
```

```

张玄素的目光中闪过一丝追忆，似是想起了什么往事。

    "江湖路远，望你……平安归来。"
```

## Act 3：下山·偶遇

### 场景：武当山下·官道·午后

**【VN节点·初入江湖】**

辞别武当，主角踏上通往江南的官道。

初夏的阳光洒在山间，官道两旁是连绵的麦田，远处偶尔可见炊烟袅袅的村庄。

```
【旁白】
离开武当的那一刻，主角忽然有种说不清的感觉。
仿佛一只被关在笼中的鸟儿，终于振翅飞向了广阔的天空。
```

他按照掌门的指引，先行前往茅山拜访。

```
【旁白】
茅山位于江南句容，距此约三日路程。
沿途可经过临安，顺道查探黑月教分舵。
```

### 场景：临安城外·酒楼·傍晚

**【VN节点·江湖百态】**

行了两日，主角来到临安城外。

他找了一间酒楼歇脚，打算打探一下消息。

```
【小二】（殷勤地）"道爷里边请！楼上雅座！"
```

主角在二楼靠窗位置坐下，点了几样酒菜。

```
【旁白】
酒楼是江湖消息最灵通的地方。
果然，四周酒客的闲聊中，夹杂着不少江湖轶闻。
```

```
【酒客甲】"听说了吗？茅山圣女下山了！"
【酒客乙】"沈霓裳？那可是江南第一美人啊！"
【酒客甲】"何止是美人，修为也是了得。据说今年才十八岁，已经是筑基三层了！"
【酒客乙】"这次五派论剑，她可是夺冠的大热门。"
```

主角端起茶杯，心中微微一动。

```
【旁白】
茅山圣女……沈霓裳……
掌门说她是故人之女，性子倔强。
看来这趟茅山之行，不会太平静。
```

就在这时，酒楼外忽然传来一阵骚动。

```
【酒客甲】（惊呼）"快看！那是……"
```

主角循声望去，只见街道尽头，一队人马正缓缓行来。

当先一人，骑着一匹雪白的骏马，身穿一袭杏黄色的道袍，腰悬一柄长剑。

她约莫十七八岁，眉如远黛，目若寒星，一头青丝用一根玉簪束起，英姿飒爽中透着一股不食人间烟火的清冷气质。

```
【旁白】
沈霓裳。
茅山圣女，武当双璧之外的江南第一天才。
雷火双修，一手「天雷诀」出神入化。
```

而在她身后，还跟着几个茅山弟子。

```
【酒客乙】（小声）"听说沈圣女是去临安查一桩案子的。"
【酒客甲】"什么案子？"
【酒客乙】（压低声音）"好像是……有人用茅山禁术害人，掌门震怒，派她下来追查。"
```

主角心中一动。

```
【主角】（内心）
茅山禁术……？
掌门让我顺道查探黑月教分舵……
这两件事，不会有关联吧？
```

### 场景：酒楼外·街道·片刻后

**【VN节点·初遇霓裳】**

主角结账下楼，正欲离开，却见那队人马已停在酒楼门口。

沈霓裳翻身下马，目光扫过四周，忽然定在了主角身上。

两人目光相接，一股奇异的感觉在主角心底升起。

```
【沈霓裳】眉头微皱。
    "你是……武当弟子？"
```

主角拱手行礼。

```
【主角】"在下武当陆季喆，奉师命前往茅山拜访。"
    "想必这位便是茅山圣女沈师姐？久仰大名。"
```

沈霓裳上下打量主角，眼中闪过一丝讶异。

```
【沈霓裳】"筑基一层？"
    "你这样的修为，也敢独自下山？"
    "武当没人了吗，派一个筑基一层的弟子出来行走江湖？"
```

她的语气中带着几分傲气和不屑。

```
【旁白】
江南第一圣女，果然名不虚传——也果然，名副其实地难缠。
```

主角面色不变，淡淡回应。

```
【主角】"师姐说的是。江湖之大，在下不过是个初出茅庐的新人。"
    "此番前往茅山，正是为了向茅山各位师兄师姐请教。"
    "若有失礼之处，还望师姐海涵。"
```

沈霓裳冷哼一声。

```
【沈霓裳】"油嘴滑舌。"
    "罢了，既然是武当弟子，便随我来吧。"
    "正好我要去查一桩案子，你若有兴趣，可以跟着看看。"
    "也算让你见识见识，什么才是真正的江湖。"
```

她翻身上马，头也不回地向前走去。

```
【旁白】
就这样，主角与沈霓裳的第一次相遇，在临安的街头落下帷幕。
一个傲气凛然的圣女，一个深藏不露的太子。
两条本不该相交的命运线，此刻开始交织。
```

**【好感度事件完成】** 沈霓裳 初遇（当前好感：20）

```
【沈霓裳】站在宅院外，神情凝重。

    "而且不止一个人。"
```

她看向主角，眼中带着几分考量。

```
【沈霓裳】"你是要跟着进去，还是在外面等着？"
    "里面可能有危险。以你的修为……"
    "算了，随你吧。反正死了也是你自己活该。"
```

```
【旁白】
这圣女说话，还真是一点儿不客气。
```

```

```

```
【旁白】
战斗，一触即发。
