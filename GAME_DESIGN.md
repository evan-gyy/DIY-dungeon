## 一、游戏定位与核心理念

### 游戏风格
「武侠暗黑地牢」× 「宝可梦式角色养成」

| 参考作品 | 借鉴元素 |
|---|---|
| 宝可梦系列 | 角色收集/养成、技能组合、进化成长 |
| 暗黑地牢（Darkest Dungeon） | 营地机制、小队管理、氛围叙事 |
| 武侠世界观 | 门派体系、内功/招式、江湖剧情 |

### 核心三大支柱
1. **角色养成**：属性成长、装备搭配、内功修炼
2. **战斗技能配合**：技能连携、门派特色招式、克制关系
3. **主线剧情**：武当/峨眉/少林/丐帮/魔教五大势力交织的江湖故事

## 三、游戏模块设计

### 模块1：角色创建界面
- 展示4种主角立绘供玩家选择（男正/男邪/女正/女邪）
- 输入角色名字
- 选择初始门派（武当/峨眉/少林/丐帮/华山/魔教）
- 新建存档并进入营地

### 模块2：营地（Camp）主界面
营地是游戏核心Hub，包含以下功能入口：

| 功能区 | 说明 |
|---|---|
| 📦 背包 | 查看/使用/装备道具 |
| ⚔️ 技能 | 查看已学技能，配置技能栏（最多4技能上阵） |
| 📊 属性 | 查看角色当前状态（HP/MP/攻/防/速/运气等） |
| 📚 学功 | 前往传功长老处学习门派技能 |
| 💬 交互 | 与营地中各门派NPC对话，推进剧情 |
| 🗺️ 出发 | 进入地图/副本（待开发） |

### 模块3：学习技能面板
每个门派传功长老处有独家技能树：

| 门派 | 技能方向 | 代表技能（示例） |
|---|---|---|
| 武当派 | 内功·剑法·太极 | 太极拳、绵掌、紫霄神功 |
| 峨眉派 | 指法·剑法·毒 | 峨眉剑法、灵蛇掌 |
| 少林派 | 拳法·棍法·金刚 | 金刚掌、罗汉拳、易筋经 |
| 丐帮 | 棍法·降龙 | 打狗棒法、降龙十八掌 |
| 华山派 | 剑法·奇术 | 华山剑法、气宗秘术 |
| 魔教 | 邪术·蛊毒·魅惑 | 乾坤大挪移、日月神教神功 |

技能学习需要：
- 达到对应门派**好感度**（通过剧情对话提升）
- 消耗**内力点数**或**江湖经验**

### 模块4：营地NPC对话
可对话NPC（初版）：
- 武当-张三丰（主线剧情触发）
- 峨眉-灭绝师太（支线剧情）
- 少林-空闻方丈（支线剧情）
- 丐帮-洪七公（支线剧情）
- 魔教-教主（隐藏/反派剧情）

对话系统支持：
- 多级选项对话树
- 好感度变化
- 触发任务/剧情事件

### 战斗数值设计原则

| 项目 | 公式 / 配置 |
|---|---|
| 基础伤害 | `max(1, ATK × powerMul - DEF × defPen)` |
| 浮动区间 | ± 10%随机 |
| 暴击 | 命中率 = crit%，倍率 = ×1.5 |
| 连击 | hit 参数控制次数，每次独立暴击判定 |
| 状态效果 | applyStatus：同类效果刷新时间取长、值取高 |
| 回血技能 | healPct × maxHp |
| 被动 | 战斗初始化时注入 playerStatus（持续999回合） |
