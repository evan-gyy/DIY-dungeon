# 第一章·重构剧本 v2：棋声、暗道与雪中人

> **v2 修订要点**（基于用户反馈）：
> 1. 墨绐青不在主角面前透露任何情报——她只是语速飞快地交代该做的事
> 2. 暗道外设**注定战败**剧情——主角没习武，打不过训练有素的密探；战败→柳清寒登场
> 3. 柳清寒人设：**外冷内热，只对主角好**——话少，冷冷一句；主角问一句她说一句
> 4. 境界体系：主角开局=普通人（仅有弈理心经被动），血脉技能不体现（避免剧透）；密探≈武林中下；柳清寒≈中流+剑心天赋

> **三重视角**：作家（文字感染力）+ 导演（场景节奏）+ 开发者（技术可行性）

---

## 📐 整体结构

```
序幕【封印之梦】    → 旁白+CG+闪白（2分钟）
第一幕【棋声】      → 对话+被动技能+教学战斗（8分钟）
第二幕【惊雷】      → 氛围骤变+暗门+逃亡（4分钟）
第三幕【暗道·逃亡】 → 注定战败+柳清寒救场（6分钟）
第四幕【雪中惊鸿】  → CG+对话+入山（5分钟）
武当日常【新起点】  → 营地+后山悼念（持续）
```

叙事结构：线性·珍珠链——宏观线性，区块内有交互自由度，收束回主线。

## 🎬 第四幕：雪中惊鸿

导演手记：全章情感高潮。脆弱→被救→初识。"雪"是核心意象——纯洁、冷冽、新的开始。
核心情感：柳清寒"知道但不能说"。等了十六年的人就在眼前，但她只会用最少的语言包裹最深的情感。

### 剑心天赋说明

**柳清寒·剑心**：
- 类型：先天天赋（非后天习得）
- 效果：剑意凝实，剑法威力+30%，所有剑系技能冷却-1
- 叙事意义：她的"冷"不只是性格，而是剑心天赋的外化——心越静，剑越利。但面对主角时的微颤，是剑心唯一的裂痕

## 📝 设计原则备忘

1. **信息精简**：每段对话只表达一个核心信息
2. **角色驱动力**：墨绐青的保护欲→只交代指令不解释；柳清寒十六年等待→少言但行动上处处照顾
3. **不剧透原则**：龙脉、太子身份、血脉力量——全部只做视觉暗示，不在文本中点破
4. **弱者叙事**：第一章主角没有逆袭，只有"被保护"和"被迫成长"。力量感留给后期
5. **少即是多**：柳清寒的话越少，情感越重。"嗯""跟上""先上山"比长篇独白更有力
